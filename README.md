# 🚗 Car Scroll Animation (Frontend Assignment)

A modern, animated hero section built using **Next.js, Tailwind CSS, and GSAP**, focusing on smooth scroll-based interactions and premium UI motion.

---

## 🌐 Live Demo

👉 https://digital-rdzw.vercel.app/

---

## 🎯 Objective

This project recreates a **scroll-driven hero animation experience**, emphasizing:

* Smooth UI transitions
* Scroll-based motion
* Clean, modern design

---

## ⚙️ Tech Stack

* **Next.js (App Router)**
* **React.js**
* **Tailwind CSS**
* **GSAP (ScrollTrigger)**

---

## ✨ Features

### 🧱 Hero Section

* Full-screen layout (above the fold)
* Modern typography and clean design
* Responsive across devices

---

### 🎬 Initial Load Animations

* Smooth entrance animations using GSAP
* Subtle fade and motion effects

---

### 🚀 Scroll-Based Animation (Core Feature)

* Car animation tied directly to scroll position
* Implemented using **GSAP ScrollTrigger**
* Uses `scrub` for smooth interpolation
* Motion based on user interaction (not autoplay)

---

### 🎨 Motion & Performance

* Uses **transform properties (translate, scale, rotate)** for better performance
* Smooth easing for premium feel
* Optimized animations with minimal layout reflows

## 🚀 Getting Started

### 1. Clone the repository

```
git clone https://github.com/Shaunak-bit/digital.git
cd your-repo
```

### 2. Install dependencies

```
npm install
```

### 3. Run the development server

```
npm run dev
```

👉 Open http://localhost:3000
